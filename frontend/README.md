# Chatapp frontend

## Startup instructions


- Provide [env variables](#env-vars)
- Install packages 

        yarn
        
- Run server 

        yarn start


## Env vars

| Variable name | Usage | 
| :---: | :---: |
| *Core* |
| REACT_APP_API_URL | Connection to api server *default:* **http://localhost:8000** | 
| *Oauth* |
| REACT_APP_FB_APP_ID | Facebook api key | 
| REACT_APP_GOOGlE_APP_ID | Google api key |
